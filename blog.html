HOME AUTOMATION

USING NODEMCU AND BLYNK

The circuit is very simple, we just need a esp8266 NodeMCU board and a relay module to control home appliances from the smartphone through the internet. We can control different home appliances from Blynk App

[+] esp8266 NodeMCU board

[+] 2 channel relay module

Download the arduino IDE (for windows).

If you are using linux(debian based) use these commands :

>> sudo apt install arduino -y

>> arduino

<<<------------------------------------------------------------------------------->>>

Download and install Blynk app on your smartphone from Google playstore. After installing the Blynk app, enter the email id and set the password to Sign Up. To create a new project in the Blynk app you have to give the project name and choose the device.

In this IoT project, we will use NodeMCU, so selected NodeMCU as your device.

After that Blynk app will send an Auth Token to the registered email id which we will need while programming the NodeMCU. Now connect the NodeMCU to your pc, fire up the arduino IDE and configure the board and port. For this IOT project, we need to install the Blynk library before programming the NodeMCU board.

Download the latest blynk library from here or install it manually from the IDE

Then to get this Blynk example sketch for Nodemcu go to: File => Examples => Blynk => Boards WiFi => NodeMCU. You will get a code similar to this:

#include
#include
// Go to the Project Settings (nut icon).
char auth[]="YourAuthToken";
// Your WiFi credentials.
// Set password to "" for open networks.
char ssid[]="YourNetworkName";
char pass[]="YourPassword";
voidsetup(){
// Debug consoleSerial.begin(9600);
Blynk.begin(auth, ssid, pass);
// You can also specify server:
//Blynk.begin(auth, ssid, pass, "blynk-cloud.com", 80);
//Blynk.begin(auth, ssid, pass, IPAddress(192,168,1,100), 8080);
}
voidloop(){
Blynk.run();
}


Make changes to the following parts in the code:

Auth Token received in Email at “YourAuthToken”.

WiFi Network Name at “YourNetworkName”.

WiFi Password at “YourPassword”

Thats all about the coding part now click on the upload button to program the NodeMCU. Wait for a few mins the program will be compiled and uploaded to the board. Now we have to create switches in the Blynk app to control the relay module.

To create a switch in Blynk App:

Open the project in the Blynk App.

Click on the “+” icon on the top.

select the Button.

Click on that button and select the output pin => D1 & Mode => Switch.
In similar way create switch with D2,D4,D5.. pins of NodeMCU. Connect the relay with the nodeMCU and power the relay taking pins Vin and Gnd from it. The relay acts as a switch in the circuit so connect the appliances accordingly with it. Thus our home automation is complete !! connect the nodeMCU to the router and the appliance can be turned on from anywhere using your smart phone.
